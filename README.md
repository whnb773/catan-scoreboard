# Catan Scoreboard

A digital scoreboard for Settlers of Catan board games.

## Files
- `index.html` - Main page
- `css/` - Styling files
- `js/` - JavaScript logic

## How to Use
1. Open `index.html` with Live Server
2. Play Catan!
3. Track scores digitally
